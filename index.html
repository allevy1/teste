Quando Surgiu o HTML?: O HTML foi inventado por Tim Berners-Lee, um físico do centro de pesquisas CERN, na Suíça. Ele surgiu com a ideia de um sistema de hipertexto na internet

O que significa hipertexto?: Hipertexto significa um texto que possui referências (links) para outros textos que podem ser acessados imediatamente.

Qual é a principal função do HTML? HTML básico - Aprendendo desenvolvimento web | MDN HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas.
